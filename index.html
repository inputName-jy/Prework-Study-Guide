HTML

  • HTML elements are case-sensitive
  • Elements are composed of an opening and closing tag represented with angle brackets
  • Closing tags are represented with a forward slash before the tag
  • Elements nested inside other elements are called child elements, making the enclosing elements the parent elements
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element within the head element defines the title for a webpage
  • The base element defines the base URL for a webpage
  • The link element connects an external resource to the HTML document
  • The style element defines the CSS styles associated with a webstie
  • The script element embeds client-sided scripts in the HTML document

CSS

  • Inline CSS refers to styling elements directly in the html file
  • Internal CSS refers to embedding an entire style sheet directly in the html
  • External CSS refers to linking an external separate CSS file to the html document using the link element in html
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript files are read from top to bottom
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript files are linked to the html document within the body
  • The JavaScript file is linked by use of the script element
  • Commenting in JavaScript is indicated by '//' for single line comments and /* as the start and */ for the ending notation for multi-line comments
  • A variable is a named container that allows us to store data in our code.
  • Conditional statements and loops can be used to control the order in which code is executed, or if it is even executed at all
  • An array is a single variable that is used to hold a group of data that is typically related in some way
  • The Array Index refers to the location number of an individual data within the array
  • In JavaScript arrays are zero-indexed and sequential, meaning the first item in any array has and index of 0
  • A 'for' loop useses predictable pattern of indicies to perform a task on all the items in an array by allowing a single block of code to be executed over and over.
